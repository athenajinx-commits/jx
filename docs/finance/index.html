Add a record

For asset categories (deposit, stock, fund, bond…) enter the item's current balance — the newest entry per item counts toward net worth, older ones become history. For income and expense categories enter the amount of each event; they add up by month.

Add a category

Assets count toward net worth (balances). Income and expenses count toward monthly cash flow (amounts).

Your categories

Rename or retype freely. A category can be deleted only when it has no records.

Export encrypted snapshot

Downloads a .finz file encrypted with AES-256-GCM (key derived from your passphrase via PBKDF2, 250,000 rounds). Opening the file shows only ciphertext — no amounts, names, or categories. The passphrase is never stored: if you forget it, the snapshot cannot be recovered.

Import snapshot

Restores a previously exported .finz file. This replaces everything currently in the app (you'll be asked to confirm).
